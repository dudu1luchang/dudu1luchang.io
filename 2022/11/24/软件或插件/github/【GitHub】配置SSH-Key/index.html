【GitHub】配置SSH-Key

【GitHub】配置SSH-Key

参考文章：https://www.jb51.net/article/241929.htm

【1】设置git的user name和email

说明：git config –list 查看当前Git环境所有配置，还可以配置一些命令别名之类的。

【2】检查是否存在SSH Key

查看C:\Users\28361.ssh目录下是否存在 id_rsa 和 id_rsa.pub文件

如果有说明存在SSH key，否则生成SSH key;

【3】获取SSH Key

进入C:\Users\28361.ssh目录；

id_rsa.pub文件中的内容就是生成的SSH key;

【4】GitHub添加SSH Key

GitHub点击用户头像，选择setting ;

新建一个SSH Key ;

取个名字，把之前拷贝的秘钥复制进去，添加就好啦。

【5】验证和修改

测试是否成功配置SSH Key

//运行结果出现类似如下
Hi xiangshuo1992! You’ve successfully authenticated, but GitHub does not provide shell access.

可能出现的问题：GitHub 无法上传 fatal: Authentication failed for ‘h

需要还需要配置一个认证
