软件安装-node

【1】进入官网下载node

node官网：https://nodejs.org/

下载旧版本：https://nodejs.org/zh-cn/download/releases/

【2】安装node

傻瓜式安装，下一步下一步；

【3】查看是否安装成功

可能出现的问题：

npm WARN config global --global, --local are deprecated. Use `–location解决方案

在node的安装目录下，找到下面2个文件里面的prefix -g替换为prefix –location=global

【4】npm切换源地址(设置为淘宝镜像)

【5】npm修改全局下载和缓存路径

注意：记得先创建下文件夹；npm-cache与npm_global

【6】npm设置用户环境变量

环境变量path中添加：D:\workSoft\nodejs\npm_global

【7】命令行安装cnpm

命令：

安装后验证：cnpm -v

npm是node官方的包管理器。cnpm是个中国版的npm

如果因为网络原因无法使用npm下载，那cnpm这个就派上用场了

一定切记切记，npm和cnpm只是下载器的不同

安装过程中出现的问题

【1】版本太新

npm ERR! ERESOLVE unable to resolve dependency tree

下载老版本的nodes【下载.msi结尾的文件】

https://nodejs.org/zh-cn/download/releases/

可以使用版本：node-v14.15.0-x64

【2】idea中不能使用哪怕npm命令

Terminal C:\Windows\System32\cmd.exe

【3】node连接不上github

npm ERR! fatal: unable to access ‘https://github.com/adobe-webplatform/eve.git/‘: OpenSSL SSL_read: Connection was reset, errno 10054

–当前无法解决
