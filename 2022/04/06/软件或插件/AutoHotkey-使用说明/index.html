AutoHotkey-使用说明

AutoHotkey使用

参考地址

下载网址：

https://www.autohotkey.com/

安装

home->modify–>Unicode 64-bit–>选择安装的文件夹–》目前使用默认安装–》Exit

创建一个项目

直接在桌面右键→→新建→→ AutoHotkey Script 即可创建一个脚本

右击–》==编辑脚本==

http://www.chinar.xin

编辑

AHK语法中： Send 是发送命令， return 是结束

运行脚本

保存后，右键文件 →→ Run Script 运行脚本

然后，当我们输入文本chinar 按下 空格键 / 回车键 / tab 键时，会自动帮我们替换为http://www.chinar.xin
