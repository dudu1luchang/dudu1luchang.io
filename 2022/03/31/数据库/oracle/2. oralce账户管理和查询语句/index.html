– 查询表的查询学习

查询表的所有数据：select * from 表明；

– 查询表中自定字段的值

数据库的操作说明 sql操作
查询表的所有数据 select * from emp
指定字段查询 select empno from emp;
select empno,ename from emp;
别名查询 select empno 员工编号 from emp;
【前提：中间不能有特殊字符;有加双引号就好】
select ename “员工姓名” from emp;
select job as 工作 from emp;
select mgr as “领导编号” from emp;【官方写法】
连接符查询 select empno||’的姓名是’||ename as 信息 from emp;
[字段名||’别名’||字段名。。。]
去重复查询 select distinct job,mgr from emp;
[去的是行重复，多行数据完全一样去重]
排序查询 select * from emp order by ename desc;
【先查再去重】【默认升序】【只要是字段名就可以】
select * from emp order by empno,ename;
【多字段排序-先排字段一，如字段一相同再按字段二排序】
字段的逻辑运算 select empno,ename,job,sal*2+100,sal+comm from emp;
【可以进行四则运算-必须要是个数字】【先查后做运算】
【计算后的字段可能为空，例如：sal+comm】
