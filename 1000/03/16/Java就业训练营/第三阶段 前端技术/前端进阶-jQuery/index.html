任务一：jQuery

jQuery介绍

美国人创建的

2006年创建的

很多框架都是基于jQuery开发的

jquery是js对象和函数的封装

jquery代码越来越少

jquery只需要一句带代码就可以实现隔行变色

jquery能做什么：

访问和操作DOM元素对象

控制页面样式

控制页面时间

扩展了新了jquery插件（鼠标放大镜效果、日历控件、地图控件、报表控件）

与Ajax技术完美结合

jquery是什么：

使用jquery可以提高开发效率

jquery是成语

js是白话文

成语是白话文的深度压缩

烽火戏诸侯与一个故事

jquery就是js

jquery能做的js都是可以做的，js能做的事情jquery不一定能做

成语能描述的事情，记叙文可不可以描述

js是大王，jQuery是js下的小王爷

jquery的优势：

体积小，压缩后只有100KB左右（jquery中有很多的方法，特别特别好yogn）

强大的选择器（遍历节点）

出色的DOM封装（创建节点、修改节点，改变节点的颜色或样式）

可靠的事件处理机制

出色的浏览器兼容性（浏览器运行效果都是一样的）

jQuery的使用

下载jquery

不要使用最新的版本，使用第二个版本

最新的东西，不是很稳定，前一个版本比较稳定一些

也可以在网络上查询哪个版本更稳定一些

是国外的网站，下载比较慢

compressed:压缩过的（代码一行显示）

production:发布

development:开发版本

拿过来就可以了，拿压缩过的发布版本就可以了

3.4.1版本，是比较稳定的（开发中我们就使用这个版本）

查看jquery-3.4.1.min.js

下载jquery-3.4.1.min.js

引入jquery

工厂函数：将DOM对象转化为jquery对象

选择器selector:获取需要操作的DOM元素对象

获取p标签中的文字

js对象和jQuery对象转换

DOM对象（HTML对象） 与jquery对象

dom对象与jquery对象自己都有一套自己的方法，不能混用

使用dom对象和jquery对象获取元素中的内容

dom对象使用 innerHTML()

jquery对象使用 html()

方法不能混用

如果dom对象想使用jquery的方法，需要将js对象转换成jquery对象

使用工厂函数($(js对象))，转换一下

jquery对象转化为js对象：jqDom.get(0);

特别注意：js对象使用innerHTML的时候，后面不要加上();

基本选择器

p标签变为红色 .css(“color”,”red”)

id选择器，具备唯一性；

一个页面有对个ID，只是第一个会生效吗？ – 只有第一个才会生效

类选择器不具备唯一性；

并集选择器：两个标签元素都会修改

交集选择器：即是又是 交在一起的元素

层次选择器

后代、子代、相邻的、

后代元素选择器（空格），忽略层级

子代元素选择器（>），只负责子元素

同辈元素选择器（~），兄弟姐妹，下面的所有兄弟元素

​ 注意：不包含上面的兄弟

相邻元素选择器（+），紧邻的兄弟h3

​ 注意是下一个紧邻的兄弟元素，不包含上一个

需要注意：script标签需要写在标签元素的下面，否则不会有效果的（因为js是解释型语言，是一行一行解释并执行的）

属性选择器

可以对属性进行一次或多次过滤（元素属性过滤）

标签中可能是有很多的属性的

属性的各种过滤特别的强悍

属性的引用使用的是[]

属性的值使用单引号‘’，使用等号连接

属性值不等于使用！

属性值以什么开头使用^(类似于正则表达式的开头)

属性值以什么结尾使用$（类似于正则表达式的结尾）

属性值包含某值使用*（包含）

多属性使用多个[]

可以和基础选择器一起使用，中间不需要加任何的符号

不理解：$(“[href!=’x’]”).css(“color”,”red”); // 全红 为什么全红

原因：p标签是不拥有这个元素的，也会将其选取出来，解决：使用标签要符合规范，属性不要乱用；标签允许使用的标签才可以使用

过滤选择器

过滤选择器重行啊

标签选择器的基础上使用过滤选择器

选择标签的偶数行元素【:even】（0,2,4。。。）

选择标签的奇数行元素【:odd】（1,3,5.。。）

选择标签的第一个元素【:first】

选择标签的最后一个元素【:last】

选择标签某行的元素【:eq(行下标)】

选择标签下标大于某行的元素【:gt(行下标)】

选择标签下标小于某行的元素【:lt(行下标)】

id等属性值不要使用特殊字符，否则无法识别，如果非要使用，需使用转义字符

转义符：\\这里需要使用两个，负责还认为有含义呢

注意：gt是大于，不是大于等于；lt是小于，不是小于等于

隐藏和显示

显示及隐藏并含有动画效果

改变元素的宽和高（带动画效果）

  • show( speed )：显示

  • hide( speed )：隐藏

  • toggle( speed )等价于show+hide : 显示的隐藏，隐藏的显示

    speed可以不使用

如果想要动画效果，需要将速度变慢

可以向方法中传入参数：

fast：快速的

slow：缓慢的

毫秒：自定义

改变元素的高（拉伸效果）

  • slideDown( speed ) ：显示
  • slideUp( speed )：隐藏
  • slideToggle( speed )等价于slideDown+slideUp

可选的 speed 参数规定隐藏/显示的速度，可以取以下值：”slow”
、”fast” 或毫秒

不改变元素的大小（淡入淡出效果）

方法 效果
fadeIn( speed ) 显示
fadeOut( speed ) 隐藏
fadeToggle( speed ) 等价于fadeIn+fadeOut动画
fadeTo( speed , 透明度 ) 方法允许渐变为给定的不透明度（值介于 0 与 1 之间）
