银行家舍入法

银行家舍入法是四舍六入。五后面有非0的数字将直接向前进位，没有数字的情况下还要看前一位是偶数还是奇数。偶数舍去，奇数进位。

总结为一句话就是：四舍六入五考虑，五后非零就进一，五后为零看奇偶，五前为偶应舍去，五前为奇要进一

错误的写法：要知道错误的原因

因为jsp是解释性语言，浏览器是一行一行读并且执行的，因为先执行的时候没有获取到这个标签元素，又在这个标签元素中加了一个监听事件，所以报空了

可以取消冒泡事件机制，传一个事件源，e.stopPropagation();
