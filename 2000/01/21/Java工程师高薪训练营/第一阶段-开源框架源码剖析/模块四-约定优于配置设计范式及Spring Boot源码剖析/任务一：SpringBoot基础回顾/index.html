约定优于配置

springBoot基于约定优于配置的；

又称按约定变成，是一种软件设计规范；

开车，靠右行驶；

java，放在src目录下；

带来一些好处，带来一些便利；

就可以省下来很多的配置；

约定优于配置：本质上是对系统、类库或框架中一些东西假定一个大众化合理的默认值（缺省值）

User实体类 User表

有一个默认约定；

如果表名叫t_user,这叫做偏移约定；

需要我们额外添加一些配置的；

总结：期望的配置和约定的配置一致，那么就可以不做任何的配置；约定不符合期待时才需要进行替换配置；

约定优于配置的好处：大大减少了配置项；
